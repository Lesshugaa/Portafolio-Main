¡Hola! Soy Thiago, un entusiasta desarrollador de software de 18 años con cinco años de experiencia en programación. Desde que descubrí mi pasión por la creación de código a los 13 años, me he sumergido en explorar y aprender diversos lenguajes y tecnologías. Mi viaje comenzó con la curiosidad por entender cómo funcionan las aplicaciones que uso a diario. A lo largo de los años, he desarrollado habilidades sólidas en HTML, CSS y Bootstrap para crear interfaces web atractivas y responsivas. Mi dominio de JavaScript me ha permitido agregar interactividad y dinamismo a mis proyectos, mientras que el uso de Git y GitHub ha sido fundamental para gestionar versiones y colaborar eficazmente en equipos de desarrollo.
